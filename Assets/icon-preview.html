Usage Instructions

1. Add to HTML <head>

<!-- Favicon --> <link rel="icon" type="image/png" sizes="32x32" href="/icons/hd-scheduler-favicon-32.png"> <link rel="icon" type="image/png" sizes="16x16" href="/icons/hd-scheduler-favicon-16.png"> <!-- Apple Touch Icon --> <link rel="apple-touch-icon" sizes="180x180" href="/icons/hd-scheduler-icon-180.png"> <!-- PWA Manifest --> <link rel="manifest" href="/manifest.json"> <!-- Theme Color --> <meta name="theme-color" content="#007BBD">
